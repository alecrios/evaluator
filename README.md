# Evaluator

## Operations

| Symbol | Operation |
|:--- |:--- |
| `+` | Addition |
| `-` | Subtraction |
| `*` | Multiplication |
| `/` | Division |
| `^` | Exponentiation |
| `%` | Modulo |
| `(`, `)` | Grouping |

## Constants

| Constant | Value |
|:--- |:--- |
| `e` | 2.71828183... |
| `pi` | 3.14159265... |

## Shortcuts

| Shortcut | Function |
|:--- |:--- |
| `CommandOrControl+Space` | Toggle the modal |
| `Esc` | Close the modal |
| `Shift+Esc` | Recenter the modal |
| `Enter` | Copy the result and close the modal |
| `Shift+Enter` | Start a new expression with the result |
| `Shift+Backspace` | Clear the expression |
